# 使用 React 和 Three.js 加载 3d模型

Three.js 文档：

https://threejs.org/docs/index.html

需要注意的是模型文件必须在 `public` 文件夹内。

## 依赖

使用到的依赖：
- "three": "^0.139.2"
- "styled-components": "^5.3.5"

因为是 TypeScript 项目所以还需要：
- "@types/three": "^0.139.0"
- "@types/styled-components": "^5.1.24"

其中 styled-components 依赖主要负责网页样式。

## 运行

进入项目文件夹内执行 `npm start`，在浏览器中打开 [http://localhost:3000](http://localhost:3000)。
